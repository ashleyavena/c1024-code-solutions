# javascript-arrays-quiz-notes

## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- What are arrays used for?
  allow you to store multiple values together in a single variable, ordering, iteration

- Describe array literal notation.
  notated by a []
  most common way to annotate is with the type[] syntax

- How are arrays different from "plain" objects?
  an array can contain an object
  objects represent "things" with characteristics (properties), array create and store lists of data in a single variable

- What number represents the first index of an array?
  [0]

- What is the `length` property of an array?
  it stores the number of items in that current array

- How do you calculate the last index of an array?
  length -1
